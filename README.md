# UnityAutoTerrainGenerator
Developing<br>

### 概要
- 地形の自動生成を主目的としたツールです。<br>
- 現在はUnityのMathf.PerlinNoise関数を使用した生成のみが可能です。

### 使い方
- Window -> AutoTerrainGenerator を選択する。
- AutoTerrainGenerator から設定値を設定し、「テレインを生成する」を押す
